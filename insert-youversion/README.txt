This plugin creates shortcodes that generate scripture references with links to the scripture on YouVersion.

== Description ==

In your post, in Visual view, click the Insert YouVersion button in the TinyMCE editor, select the version and book, enter the chapter and verse and click Insert. A shortcode in the format below will be added.

[insert-youversion ref=niv.gen.1.1]

You can force the link to open in the same window by selecting "Same Window" under "Target" when adding a shortcode, or by adding "target=self" to the end of the shortcode.

== Installation ==

= Using The WordPress Dashboard =

1. Upload the `insert-youversion` folder to the `/wp-content/plugins/` directory or install through The WordPress Dashboard.
2. Activate the plugin through the 'Plugins' menu in WordPress.
